The Cube Game

A 3D mobile cube with a new minigame on each face. Play daily, earn scores, and compete on global leaderboards in quick, addictive challenges.

JS • HTML • React Native • Game Dev
View on Github

Using JS and React Native I’m developing a 3D interactive cube where each face unlocks a different minigame. Players can rotate the cube and take on daily challenges ranging from quick reflex tests like the Ball Game to social mind games such as The Opposite Game. Each minigame can only be played once per day, giving players a score that contributes both to their individual game result and to a combined total across all cube faces. With global leaderboards, the Cube Game blends fun, addictive mechanics with competitive replay value, encouraging players to return daily and climb the rankings.
